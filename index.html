Installation

  1. Purchase and download the script package (.lua and resources).
  2. In REAPER, open Actions → Show Action List → ReaScript → Load.
  3. Select the main script file (e.g., FXD_Vertical FX list.lua).
  4. REAPER indexes it; search the script name and assign a shortcut.
  5. Optional: Add to a toolbar for quick access.
