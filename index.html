How it works

  1. Install

    Download the script file after purchase. Drag it into Reaper or use the Action List to import.

  2. Bind a shortcut

    Open the Action List, search for the script, and assign your favorite key.

  3. Go faster

    Use on selections, items, or time ranges. The script adapts to your context.

Requirements

  • REAPER 6.0+ (tested on 6.x / 7.x)
  • macOS 12+ or Windows 10+
  • SWS Extension recommended
  • ReaImGui 0.9.3+ (bundled in script)

Installation

  1. Purchase and download the script package (.lua and resources).
  2. In REAPER, open Actions → Show Action List → ReaScript → Load.
  3. Select the main script file (e.g., FXD_Vertical FX list.lua).
  4. REAPER indexes it; search the script name and assign a shortcut.
  5. Optional: Add to a toolbar for quick access.
